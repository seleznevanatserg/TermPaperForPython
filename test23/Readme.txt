Программа берёт файл настроек (settings.json), считывает оттуда максимальное занчение MaxId для номера будующеё заметки, прибавляет к нему +1 и перезаписывает в файл. 
MaxId используется для уникального номера заметки и названия файла.
По итогу программа создаёт новый файл (*.json).
Этот кусок является частью функции по созданию новых заметок.
